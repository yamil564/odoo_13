Sales Barcode Scanning

This module adds support for barcodes scanning to the Sales Management & will help you in sale order to make your process faster.

Features
  • Scan and add products effortlessly.
  • Search existing products by barcode & internal reference of the product.
  • Use barcode to add products in quotation and sale orders.
  • Reduces operational time consumed in manual entry of products.
  • Reduce probability of human error while adding the product to sale orders.
  • A warning message pops up each time when a wrong barcode is scanned.
Sale Order View

User can easily scan a product which will be added in order line.

Warning message for wrong barcode scanning

A warning message pops up each time when a wrong barcode is scanned.
